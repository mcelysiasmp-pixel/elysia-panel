# themes/

Emplacement prévu pour des thèmes du Dashboard packagés séparément
(feuilles de variables CSS / configuration Tailwind alternatives),
installables via le Marketplace.

**État actuel** : non implémenté. Le Dashboard supporte déjà un mode
sombre/clair (`next-themes`, voir
[`dashboard/src/components/theme-provider.tsx`](../dashboard/src/components/theme-provider.tsx))
mais pas encore de thèmes tiers chargés dynamiquement.
